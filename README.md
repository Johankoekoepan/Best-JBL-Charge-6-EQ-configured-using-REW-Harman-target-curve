# Best-JBL-Charge-6-EQ-configured-using-REW-Harman-target-curve

The reason I'm making this post is because I tried looking for an EQ online that'll make the JBL Charge 6 sound as good as it can but found nothing and want to share my EQ for anyone to use if you like it :)

I know nothing about audio tuning except a yt video I watched showing how to do this and me testing and tweaking a LOT of times to try and make my JBL Charge 6 speaker sound as good as it possibly can.

So if anyone with expertise can show/tell me how I can make it even better or if I did anything wrong im open to suggestions.

That said, it sounds AMAZING and I'm quite surprised how bad JBL's default sound signature sounds compared to making it replicate the harman target curve as close as possible without distortions.

There are 2 EQ profiles, the one has 11PK's and the other 20PK's, the one with 20PK's is a little bit closer to the harman curve but has more gain so it MIGHT distort at extremely high volumes but I haven't had any distortions myself so if you want to be more sure there wont be any distortions use the 11PK EQ. I personally use the 20PK EQ and haven't had any distortions but also haven't used my speaker at max volume. (I think JBL automatically limits the base above 60% anyway to make sure the speaker doesn't get damaged so ya.)

I used the best budget consumer mic to do sweep tests of my speaker (sure it might not be the BEST but I feel like it's really good and atleast good enough): Fifine T669 Pro 3 RGB Cardioid Condenser Microphone
