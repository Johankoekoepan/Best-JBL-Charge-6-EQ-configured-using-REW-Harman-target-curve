# Best-JBL-Charge-6-EQ-configured-using-REW-Harman-target-curve

The reason I'm making this post is because I tried looking for an EQ online that'll make the JBL Charge 6 sound as good as it can but found nothing and want to share my EQ for anyone to use if you like it :)

The difference from JBL's default sound signature isn't extremely noticible on all songs but a lot of songs are CLEARLY a LOT better, less tinny and sounds fuller.

I know nothing about audio tuning except a yt video I watched showing how to do this and me testing and tweaking a LOT of times to try and make my JBL Charge 6 speaker sound as good as it possibly can.

So if anyone with expertise can show/tell me how I can make it even better or if I did anything wrong im open to suggestions.

That said, it sounds AMAZING and I'm quite surprised how bad JBL's default sound signature sounds compared to making it replicate the harman target curve as close as possible without distortions.

There are 3 EQ profiles, the one has 11PK's and the other 20PK's, the 20PK is more precise than the 11PK EQ and the newer 20PK EQ is even more precise than the old 20PK. I personally use the old 20PK EQ and haven't had any distortions but also haven't used my speaker at max volume. (I think JBL automatically limits the base above 60% anyway to make sure the speaker doesn't get damaged so ya.)

I used the a very good budget consumer mic to do sweep tests of my speaker (sure it might not be the BEST but I feel like it's really good and atleast good enough for amazing results): Fifine T669 Pro 3 RGB Cardioid Condenser Microphone


Different EQ's:

•  11PK EQ: smooth match to harman target curve.

•  OLD 20PK EQ: a lot closer (more precise) match to the harman curve with a little bit lower vocals than the newest 20PK.

•  NEW 20PK EQ: closest to the harman curve. (Technically the best i guess but I personally still prefer the old 20PK EQ)
